Shirt Manager
NFC interaction
One of the features I wanted was the ability to intelligently find shirts quickly, so I decided to use NFC. This means I could sew laundry-proof NFC tags onto my shirts for easy and quick identification.
My shirts
They're my most prized possessions. I wanted to create a system for me to track every single shirt I own, a bit like an inventory system but for shirts.
Fit for me
The app was specifically designed to fit my needs, such as parameters of shirts like body fit (which I care about a lot). It had one purpose, to track my shirts.

Manage Shirts
The app sends a request to an API which returns JSON data depending on what the app requests via its data parameters. This is then returned and handled by the app and shown in the interface. Sorting and filtering happens at the API level with the SQL queries to the MySQL database. Cards can then be clicked on to see more details and detailed images.
Smart Viewing
Intelligent open shirts via NFC. Using NFC support with Android, whenever an NFC tag with a shirt data type is detected, the app is automatically opened and the respective shirt is displayed. From this, you can edit details, mark it as washed, or a variety of different quick functions.
