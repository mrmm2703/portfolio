Mental Screen
The idea
The app records app usage and screen time everyday, which is then sent to an API which connects to a MySQL database storing all this data. Students are then put on leaderboards and classes to compete against each other and for prizes.
Social media detox
Mental Screen (not a final name) was created with States of Minds, a mental health organisation, to help students get less screen time to better their mental health.

Screen time tracking
Mental Screen automatically shows you your social media usage based on daily, weekly and monthly usage. These statistics are then transferred over to a database via an API call and stored inside a MySQL database.
UI Navigation
There was a lot of data sitting on the MySQL database ready to be accessed and visualised. I wanted to get as much of this information out as possible, which inevitably made navigation a challenge. So I stuck to a combination of ViewPagers and a navigation drawer to show all this data.
Leaderboards
There had to be a way to show users their points against other users in their teams, year groups, schools and globally. It was the best way to show as much data as possible without overwhelming the user with data they don't want to see. Users can then further choose if they'd like to compare daily, weekly or monthly points within the chosen category.
Competitions
Then there had to be a way for admins and teachers to set competitions for specific categories of people, depending on year group and school. This all was handled through the SQL queries, by ensuring they were as concise and specific as possibe to allow for filtering of data.
